## 📌 О проекте

Полноценное решение для интеграции и управления устройствами умного дома от Дом.ru (домофоны, калитки, шлагбаумы, видеокамеры). Проект состоит из трёх основных частей:

1. **Современная веб-панель (React Dashboard):** Адаптивный интерфейс для просмотра камер, открытия дверей, управления расписаниями авто-открытия, просмотра истории событий и снапшотов.
2. **TypeScript API Клиент (`domru-api`):** Типизированная библиотека для программного взаимодействия с экосистемой Дом.ru.
3. **SIP-сервер:** Встроенный SIP User-Agent, перехватывающий звонки домофона для авто-открытия дверей и создания снапшотов камер.

> [!IMPORTANT]
> Проект не аффилирован с провайдером Дом.ru. Используйте его на свой страх и риск. Храните учётные данные только в безопасном окружении.

## ✨ Возможности

### 🖥️ Веб-интерфейс
- **Просмотр видео в реальном времени:** HLS-стриминг через FFmpeg прокси + WebRTC через go2rtc для минимальной задержки.
- **Управление доступом:** Удалённое открытие дверей и калиток в один клик.
- **Авто-открытие по расписанию:** Гибкие недельные расписания для жильцов, гостей и курьеров с ежедневным сбросом счётчика открытий.
- **Временные пропуска:** Создание карточек «Быстрый доступ» с ограниченным числом открытий и временем действия.
- **Снапшоты при звонке:** Автоматическое сохранение JPEG-снимков камер при каждом входящем SIP-звонке.
- **История событий:** Журнал звонков, открытий дверей с привязкой к снапшотам и источнику открытия (вручную, по расписанию, через Алису).
- **Интеграция с Алисой:** Yandex Smart Home (открытие дверей как замков, камеры) + Yandex Dialogs (голосовые сценарии: «Алиса, жду курьера»).
- **Real-time обновления (WebSocket):** Мгновенная синхронизация состояния через Socket.IO — индикаторы авто-открытия, SIP-логи, уведомления о звонках обновляются без перезагрузки страницы.
- **Умное время:** Все расписания работают в московском часовом поясе (MSK, UTC+3) независимо от региона размещения сервера.
- **Адаптивный дизайн:** Тёмная и светлая тема, мобильная и десктопная версии.
- **Режим песочницы (Demo):** Тестирование интерфейса без реальных учётных данных.
- **Безопасность:** Middleware `requireDomruAuth` на всех приватных маршрутах.

### ⚙️ API Клиент (`domru-api`)
Библиотека в `src/domru-api/` предоставляет:
- Автоматическое получение и обновление токенов доступа.
- Кеширование запросов.
- Встроенный `EventEmitter` для отслеживания изменений токенов и ошибок.
- Методы: список мест, устройств, камер, открытие дверей, снапшоты (JPEG), видеопотоки, SIP credentials.

Подробная документация: [src/domru-api/README.md](src/domru-api/README.md).

## 🚀 Установка и запуск

**Требования:** Node.js v18+

### Локальная разработка

```bash
git clone https://github.com/DKheruvimov/domru-integration.git
cd domru-integration
npm install
npm run dev
```
*Приложение будет доступно по адресу: http://localhost:5173 (Vite dev server проксирует API на бэкенд)*

### Сборка для Production

```bash
npm run build    # Vite (фронтенд) + esbuild (бэкенд → dist/server.cjs)
npm start        # Запуск продакшн-сервера
```

Сервер слушает `0.0.0.0` на порту из `server/config.ts` (по умолчанию `3000`). Переопределение:
```bash
PORT=8080 npm start
```

### Docker

```bash
docker build -t domru-integration .
docker compose up -d
```
Контейнер маппит порт `3100` (хост) → `3000` (контейнер). Данные сохраняются в volume `./data`.

> [!TIP]
> **Развертывание на публичном сервере:** Рекомендуется спрятать сервер за Reverse Proxy (TurboFlare, Cloudflare) в режиме `Full Strict`. В проекте встроен `app.set("trust proxy", true)` для корректного определения IP клиентов.

## 📖 Краткий пример использования API (`domru-api`)

```typescript
import { DomruClient } from "./src/domru-api";

const client = new DomruClient({
  login: "ваш_логин",
  password: "ваш_пароль"
});

await client.authenticate();

const places = await client.getSubscriberPlaces();
if (places.length > 0) {
  const placeId = places[0].place?.id || places[0].id;
  const devices = await client.getDevices(placeId);
  
  // Открыть первую доступную дверь
  const intercom = devices.find(d => d.allowOpen);
  if (intercom) {
    await client.openDoor(placeId, intercom.id);
    console.log("Дверь открыта!");
  }
}
```
