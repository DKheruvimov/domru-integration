# API Внешних Модулей (External Modules API)

Внешние модули — это сторонние скрипты или приложения (например, Telegram-боты, n8n, Home Assistant), которые могут взаимодействовать с Ядром (Dom.ru Integration Hub) для получения событий с домофона и отправки команд (например, открытия двери).

## Авторизация (Токен)
Для работы с API вашему модулю нужен **Токен**. 
Токен можно сгенерировать в веб-интерфейсе Ядра (раздел "Настройки" -> "Внешние модули"). 

Авторизация передается в каждом HTTP-запросе одним из способов:
1. Через заголовок: `Authorization: Bearer <ваш_токен>` (рекомендуется)
2. Через query-параметр: `?token=<ваш_токен>`

---

## 1. Получение событий (Webhook / WebSocket)

Модуль может получать события от Ядра (например, когда кто-то звонит в домофон) двумя способами: через Webhooks или через WebSockets. Модуль сам решает, какой способ ему удобнее.

### Способ А: Настройка Webhook (HTTP POST)
Если вы хотите, чтобы Ядро отправляло POST-запросы на ваш сервер при наступлении событий, вам нужно зарегистрировать свой Webhook URL в Ядре.

**Запрос (от Модуля к Ядру):**
`POST /api/modules/me/webhook`

**Заголовки:**
`Authorization: Bearer <ваш_токен>`
`Content-Type: application/json`

**Тело:**
```json
{
  "webhookUrl": "https://ваш-домен.com/api/webhook"
}
```

**Формат события (от Ядра к Модулю):**
Ядро будет отправлять POST-запросы на ваш `webhookUrl` с таким телом:
```json
{
  "event": "incoming_call",
  "payload": {
    "callId": "...",
    "intercomId": "...",
    "timestamp": 1690000000000
  }
}
```

### Способ Б: Подключение по WebSocket
Если вашему скрипту проще держать постоянное соединение (например, локальные Python-скрипты), вы можете подключиться по протоколу WebSocket.

**URL для подключения:**
`ws://<адрес_ядра>/api/modules/ws?token=<ваш_токен>`

После успешного подключения Ядро начнет транслировать все события домофона в виде JSON-сообщений.

---

## 2. Действия и Команды

Модуль может отдавать команды Ядру, например, открыть дверь домофона или сделать фото с камеры.

### Открыть дверь (Action Open)
**Запрос:**
`POST /api/modules/actions/open`

В заголовке передаем `Authorization: Bearer <ваш_токен>`.
В случае успеха вернется JSON `{ "success": true }`.
